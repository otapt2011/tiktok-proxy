🎵 TikTok Scraper API

Professional REST API for public TikTok profile data – ready for any project

GET Full user profile

/api/user/:username

Returns complete webapp.user-detail object (bio, stats, avatar, etc.).

:username – TikTok username (without @)

Example: /api/user/jafernegery

GET Only statistics

/api/user/:username/stats

Returns follower, following, heart, and video counts.

Example: /api/user/jafernegery/stats

GET Recent videos

/api/user/:username/videos?limit=12&cursor=0

Returns list of user's recent videos with metadata (play count, likes, comments, etc.).

limit (optional, default 12) – number of videos to return
cursor (optional, default 0) – pagination cursor (currently limited to first page)

Example: /api/user/jafernegery/videos?limit=5

POST Batch fetch users

/api/batch

Fetch multiple profiles in one request. Body must be JSON:

{"usernames": ["jafernegery", "lorenagray"]}

Returns array of results (success + data or error).

Example using curl:
curl -X POST https://tiktok-proxy-theta.vercel.app/api/batch -H "Content-Type: application/json" -d '{"usernames":["jafernegery"]}'

🔁 Legacy endpoints (backward compatible)

GET /api/tiktok?username=... – Lightweight profile (original)
GET /api/tiktokFull?username=... – Full user detail (original)

These will still work but new projects should use the RESTful endpoints above.

⚡ Serverless on Vercel | CORS enabled | No authentication required
