🎵 TikTok Proxy

Serverless API to fetch public TikTok profile data.

📡 Endpoints

GET /api/user/:username

Full user detail (bio, stats, avatar, etc.)

GET /api/user/:username/stats

Only follower/following/heart/video counts

GET /api/user/:username/videos?limit=12

Recent videos with metadata

POST /api/batch

Batch fetch multiple users – body {"usernames":["user1","user2"]}

GET /api/tiktok?username=...

Legacy lightweight profile

GET /api/tiktokFull?username=...

Legacy full user detail

🔑 Authentication

All endpoints require an X-API-Key header. Use the same key you set as API_SECRET_KEY in Vercel.
