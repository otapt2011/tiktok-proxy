🎵 TikTok Proxy API

Serverless endpoints to fetch public TikTok profile data
GET

📘 Lightweight Profile

/api/tiktok?username=jafernegery

Returns selected profile fields: username, display_name, avatar_url, followers, following, total_likes, total_videos.
Use case: Quick avatar, follower counts, lightweight scraping.

🔍 Try it →

GET

📦 Full User Detail

/api/tiktokFull?username=jafernegery

Returns the complete webapp.user-detail object from TikTok's internal data, including userInfo.user, userInfo.stats, and every available field.
Use case: Full data extraction, deep analysis, bio, links, and more.
